Lexer

This program uses .cci and .ccx files as the input. It then parses the file into categories and writes it to an output file. The 8 categories are

  • Comment
  • String
  • Keyword
  • Character Literal
  • Operator
  • Numeric Literal
  • Identifier
  • Unknown

Usage: ./lexer input_file output_file
